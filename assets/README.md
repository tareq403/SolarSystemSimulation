## Image Sources

You can download high-quality images from the following sources:

### NASA Image Library
- **URL**: https://images.nasa.gov/
- Search for planet names (e.g., "Mercury", "Jupiter")
- Download high-resolution images

### Solar System Scope Textures
- **URL**: https://www.solarsystemscope.com/textures/
- Free textures for all planets and moons
- Download the texture maps

### NASA Science Solar System
- **URL**: https://science.nasa.gov/solar-system/
- Official NASA planetary images

### Wikipedia Commons
- Search for each planet/moon name
- Look for public domain or Creative Commons images

## Recommended Image Sizes

- **Sun**: 512x512px (or larger)
- **Large planets** (Jupiter, Saturn): 256x256px
- **Medium planets** (Earth, Neptune, Uranus): 128x128px
- **Small planets** (Mercury, Venus, Mars): 128x128px
- **Moons**: 64x64px to 128x128px

## Image Requirements

- Format: PNG (with transparency if desired) or JPG
- Square aspect ratio preferred
- Reasonable file sizes (under 1MB per image)

## Notes

- If images are missing, the application will fall back to colored circles
- The application still works without images, but visuals will be less realistic
- For best results, use actual NASA photographs or rendered textures

## Quick Download Guide

1. Visit https://www.solarsystemscope.com/textures/
2. Download textures for all planets and Sun
3. Resize images to recommended sizes
4. Place them in the appropriate directories
5. Rename to match the filenames above (lowercase, no spaces)

Alternatively, you can use simple colored circles by not providing images - the application handles this automatically with fallback colors.
